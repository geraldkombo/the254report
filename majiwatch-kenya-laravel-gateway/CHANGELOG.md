# Changelog

## 0.2.0 — 2026-05-01
- Mobile-first dashboard navigation and layout.
- Water points delivered as PostGIS vector tiles (`/api/tiles/waterpoints/...`) for fast map rendering.
- Offline fallback map style + installable manifest + service worker caching.
- Same-origin deployment pattern via `/api/*` and `/reports/*` reverse proxy.
- Signed oracle feed endpoint (`/oracle/water-quality/latest`) for smart-contract integrations.
- Expanded datasource catalog (35+ entries) and ingestion run artifact tracking.
