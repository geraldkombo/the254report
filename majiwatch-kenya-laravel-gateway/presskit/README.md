## Suggested description (short)
MajiWatch Kenya monitors Kenya’s water and sanitation sector at county level. It combines public datasets into nightly scores (0–100), watch/warning/emergency alerts, and automatic PDF briefs for rapid response and oversight.

## Suggested description (long)
MajiWatch Kenya is an open-source, production-ready decision infrastructure for SDG 6 in Kenya. It provides a desktop and mobile-first dashboard, a documented REST API, and nightly computation of six sub-scores (water access, sanitation coverage, water quality, utility performance, governance, climate resilience) plus a composite score. When thresholds are breached, the platform generates auditable emergency briefs to accelerate coordination and response.
